# Getting Started with Flux Theme

Welcome! This guide walks you through setting up Flux Theme for the first time.

## Step 1: Initial Configuration

Open **Flux Theme Settings** and configure your preferences.

## Step 2: Set Up Users and Roles

Assign roles to your team members using the CAPS permission system.

## Step 3: Start Using

Navigate to the main workspace and begin creating records.
