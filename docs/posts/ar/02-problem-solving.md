## يعمل مع ERPNext الحالي لديك

Flux Theme لا يستبدل ERPNext — بل **يُوسّعه**. ثبّته بجانب إعدادك الحالي:

```bash
bench get-app https://github.com/sarapil/flux_theme
bench --site your-site install-app flux_theme
```

كل بياناتك وسير عملك وتخصيصاتك الحالية تبقى كما هي.
